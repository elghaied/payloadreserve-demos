## Installation

```bash
npm install payload-reserve
```

## Quick Setup

Add the plugin to your `payload.config.ts`:

```typescript
import { buildConfig } from 'payload'
import { reservePlugin } from 'payload-reserve'

export default buildConfig({
  plugins: [
    reservePlugin({
      resources: ['specialists'],
    }),
  ],
  // ... rest of config
})
```

## What Gets Added

After installing, the plugin automatically adds:

- **Reservations** collection — stores all bookings with status, timestamps, and linked resource
- **Schedules** collection — defines available time windows per resource
- **Public REST API** — `/api/reserve/availability`, `/api/reserve/create`, etc.
